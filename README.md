# Image Processor (C++)

Консольное приложение для обработки изображений в формате BMP.  
В приложении реализован набор классических фильтров, которые можно применять последовательно через аргументы командной строки.

Используется стандарт C++23.

---

## Возможности

- Чтение и сохранение изображений в формате BMP
- Последовательное применение нескольких фильтров
- Расширяемая архитектура (каждый фильтр - отдельный класс)

### Поддерживаемые фильтры

| Фильтр | Аргументы | Описание |
|------|----------|----------|
| `-crop` | `width height` | Обрезка изображения, cохраняется левый верхний угол |
| `-blur` | `sigma` | Гауссово размытие |
| `-edge` | `threshold` | Выделение границ |
| `-gs` | - | Оттенки серого |
| `-neg` | - | Негатив |
| `-sharp` | - | Резкость |
| `-vortex` | `strength` | Вихревое искажение |
| `-waves` | `strength` | Волновое искажение |

Если фильтры не указаны, изображение копируется без изменений.

## Сборка

### Требования

* Компилятор с поддержкой C++23
* CMake $\geq$ 3.10

### Сборка проекта

```bash
git clone <url-репозитория>
```

```bash
cd image_processor
mkdir -p build
cmake ..
make
```

---

## Запуск

Общий формат запуска:

```bash
./image_processor <input.bmp> <output.bmp> [filters...]
```

### Примеры

Применить гауссово размытие и затем повысить резкость:

```bash
./image_processor input.bmp output.bmp -blur 2.0 -sharp
```

Обрезать изображение и перевести в оттенки серого:

```bash
./image_processor input.bmp output.bmp -crop 300 300 -gs
```

Применить волновое искажение:

```bash
./image_processor input.bmp output.bmp -waves 0.1
```
